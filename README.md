# Abdullah Haleem — Portfolio Website

A modern, responsive single-page personal portfolio for **Abdullah Haleem**, a web developer and YouTube content creator. Built with plain HTML, CSS, and JavaScript — no frameworks, no build step.

🔗 **Live:** open `index.html` in any modern browser.

## Features

- **Responsive design** — looks great from mobile to desktop
- **Dark / light mode toggle** with preference saved to `localStorage`
- **Animated hero** with a typing role ticker (Developer · YouTuber · Creator · Problem Solver)
- **Scroll-reveal animations** powered by `IntersectionObserver`
- **Active nav highlighting** that tracks the section in view
- **Animated skill bars** and **circular SVG progress charts**
- **Filterable portfolio grid** (All / Web / YouTube)
- **Embedded YouTube videos** section
- **Contact form** with client-side validation and a success message
- **Mobile hamburger menu**

## Sections

Home (Hero) · About · Skills · Services · Portfolio · Videos · Contact · Footer

## Tech Stack

- **HTML5** — semantic markup
- **CSS3** — custom properties, flexbox/grid, animations, fully responsive
- **Vanilla JavaScript** — no dependencies
- **Google Fonts** — Poppins + Inter
- **Font Awesome 6** — icons (via CDN)

## Getting Started

No installation or build tools required.

1. Clone the repository:
   ```bash
   git clone https://github.com/Abdullah4589/portfolio-website.git
   ```
2. Open `index.html` in your browser — or serve it locally:
   ```bash
   # Python
   python -m http.server 8000
   # then visit http://localhost:8000
   ```
